# 🤖 Recruiter AI

AI-Powered Talent Acquisition & Interview Intelligence System

---

## 🚀 Overview

Recruiter AI is a full-stack AI-powered recruitment system that automates resume screening, job description analysis, and candidate evaluation through semantic intelligence.

It reduces manual hiring effort by transforming unstructured recruitment data into structured, machine-readable insights using AI.

---

## 🎯 Problem Statement

Traditional hiring systems face major limitations:

- Keyword-based filtering misses qualified candidates
- Manual resume screening is time-consuming
- Interview evaluation is subjective and inconsistent
- Lack of structured candidate intelligence and scoring

---

## 💡 Solution

Recruiter AI solves these issues by introducing an AI-driven recruitment pipeline that:

- Understands job descriptions semantically
- Matches resumes based on contextual meaning, not keywords
- Automates interview simulation using structured logic
- Generates objective hiring recommendations

---

## 🏗️ System Architecture

The system is built using a modular full-stack architecture:

- **Frontend Layer** → React-based UI for recruiters and candidates  
- **Backend Layer** → Node.js (Express) API handling logic and orchestration  
- **AI Layer** → Google Gemini API for semantic understanding  
- **Interaction Layer** → Web Speech API for voice-based interviews  

### Workflow Pipeline:

Job Description → AI Parsing → Resume Matching → Interview Simulation → Final Evaluation

---

## ⚙️ Core Features

### 📄 Job Description Analyzer
Extracts structured requirements such as skills, experience level, and domain.

### 🧠 Resume Matching Engine
Performs semantic comparison between job requirements and candidate resumes.

### 🎤 AI Interview System
Conducts structured 5-step interviews using voice interaction.

### 📊 Evaluation Engine
Generates final candidate score and hiring recommendation.

---

## 🧩 Engineering Highlights

- Modular full-stack architecture with clear separation of concerns  
- Secure backend-only AI processing for API protection  
- Structured JSON data flow between system components  
- Fallback logic to ensure system reliability during API failure  
- Designed with scalability and future ATS integration in mind  

---

## 🔁 System Workflow

1. Recruiter inputs job description  
2. AI extracts structured job requirements  
3. Candidate uploads resume  
4. System performs semantic matching  
5. Candidate undergoes AI-driven interview  
6. Final evaluation report is generated  

---
