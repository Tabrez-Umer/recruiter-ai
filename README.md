# 🤖 Recruiter AI

AI-Powered Talent Acquisition & Interview Intelligence System

---

## 🚀 Overview

Recruiter AI is an intelligent hiring system that automates resume screening and interview evaluation using AI.

It helps recruiters:
- Understand job requirements
- Match resumes intelligently
- Conduct structured AI interviews
- Generate final hiring reports

---

## 🎯 Problem

Traditional hiring systems are slow and biased:
- Keyword-based resume filtering misses good candidates
- Manual interviews are inconsistent
- Recruiters spend too much time screening

---

## 💡 Solution

Recruiter AI automates the entire hiring pipeline using AI:

✔ Semantic job description parsing  
✔ AI-based resume matching  
✔ Voice-enabled interview system  
✔ Automated scoring and reporting  

---

## ⚙️ Features

### 📄 Job Analysis
- Extracts structured job requirements using AI

### 🧠 Resume Matching
- Compares resume with job description
- Generates match score (0–100)

### 🎤 AI Interview System
- 5-question structured interview
- Voice-based interaction (browser APIs)

### 📊 Final Evaluation
- Candidate scoring system
- Hiring recommendation output

---

## 🛠️ Tech Stack

- React 18
- Node.js (Express)
- Gemini AI API
- JavaScript
- Web Speech API
- HTML / CSS

---

## 🧠 How It Works

1. Recruiter enters job description
2. System extracts structured requirements
3. Candidate uploads resume
4. AI calculates match score
5. Candidate goes through interview
6. Final report is generated
